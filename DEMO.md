# 60-Second Demo

```bash
npm install -g openclaw@2026.4.5 clawpowers-agent
clawpowers init
clawpowers status
clawpowers run "Reply with READY and a one-line summary of the active runtime."
```

What this demonstrates:
- stock OpenClaw runtime present
- ClawPowers plugin installed
- skills synced into the OpenClaw environment
- wrapper command successfully routes a task through OpenClaw
